Comandos de terminal

  • Ls

    • Te permite ver el contenido del archivo donde estás

  • Cd {ruta}

    • Cambia al directorio al nombre

  • Cd ..

    • Retrocede un nivel en la estructura de directorios

  • Mkdir {nombre}

    • Crea un nuevo directorio

  • rmdir {nombre}

    • Elimina un directorio vacio

  • rm {archivo}

    • Elimina un archivo

  • rm -r {carpeta}

    • Elimina una carpeta y su contenido

  • Cat {archivo}

    • Muestra el contenido de un documento

  • Head {archivo}

    • Muestra las primeras 10 líneas de un documento

  • Tail {archivo}

    • Muestra las últimas 10 líneas de un documento

  • Clear

    • Limpia la terminal
