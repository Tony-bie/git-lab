Preguntas

  1. ¿Qué es el DOM?

    El DOM (Document Object Model) es una representación en forma de árbol de los elementos de un documento HTML/XML. Permite interactuar con la estructura, contenido y estilos mediante JavaScript. [MDN]

  2. ¿Qué es un nodo?

    Un nodo es cualquier objeto dentro del DOM, como elementos HTML, atributos o texto. Ejemplo: <p>Hola</p> es un nodo de tipo elemento. [MDN]

  3. ¿Cómo se accede a un elemento del DOM?

    Usando métodos como getElementById() o querySelector(). Ejemplo:

    document.getElementById('miId');
  4. ¿Cómo se modifica un elemento del DOM?

    Cambiando propiedades como textContent o innerHTML. Ejemplo:

    document.querySelector('p').textContent = 'Nuevo texto';
  5. ¿Cuál es la diferencia entre getElementby* y querySelector*?

    getElementById() retorna un solo elemento por ID, mientras que querySelector() usa selectores CSS y retorna el primer elemento que coincida. [MDN]

  6. ¿Qué es un evento?

    Una acción o ocurrencia (e.g., hacer clic, mover el mouse) que JavaScript puede detectar y manejar. [MDN]

  7. ¿Qué es un listener?

    Una función que "escucha" un evento y ejecuta código cuando este ocurre. Ejemplo:

    boton.addEventListener('click', () => alert('¡Hola!'));
  8. ¿Qué es un callback?

    Una función pasada como argumento a otra función, que se ejecuta después de un evento o proceso. [MDN]

  9. ¿Qué es un objeto de evento?

    Un objeto que contiene información sobre un evento (e.g., coordenadas del mouse, tecla presionada). Se pasa automáticamente a los listeners. [MDN]

Conclusiones

Experiencia general

La experiencia fue enriquecedora, ya que permitió combinar el uso de herramientas de IA para resolver problemas de programación de manera eficiente.

¿Cuándo y para qué usarías ChatGPT y cuándo Copilot?

Usaría ChatGPT para obtener explicaciones detalladas y ejemplos de código, y Copilot para autocompletar y generar código rápidamente en el contexto del proyecto.

Ventajas de usar AI para programar

Incrementa la productividad, reduce el tiempo de desarrollo y ayuda a aprender nuevas técnicas.

Desventajas de usar AI para programar

Puede generar código incorrecto si no se verifica, y puede fomentar la dependencia excesiva de las herramientas.
