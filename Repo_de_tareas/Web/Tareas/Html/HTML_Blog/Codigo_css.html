Código de CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para dar estilo y presentación a documentos de HTML. Permite que el contenido del HTML y la personalización se separen, evitando generar conflictos con el código.

¿Qué son los selectores y sus tipos?

Son patrones utilizados para aplicar estilos a elementos específicos.

Tipos:

  • Selectores por tipo

    Seleccionan todos los elementos de un tipo en particular.

  • Selectores por clase

    Aplican estilos a elementos con una clase específica.

  • Selectores por id

    Aplican estilos a un único elemento con un identificador.

Diferencias entre selectores

  • Tipo: Se aplica a todos los elementos de un mismo tipo.

  • Clase: Se aplica a cualquier elemento con la clase específica.

  • Identificador: Se aplica a un único elemento.
